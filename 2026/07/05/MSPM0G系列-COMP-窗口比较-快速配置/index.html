    概述

    本篇讲解如何快速配置 MSPM0G 系列 MCU 的 COMP 工作在窗口比较器的模式，有关具体原理的讲解请移步至MSPM0G系列 COMP-特性讲解。

    sysconfig配置

    主要配置项

    COMP/
|——Basic Configuration/
|   |——Reference Voltage Generator Configuration 
|——Advanced Configuration

    注：COMP 需要配置两次，两次配置有细微差别，请仔细阅读后文。

    将对下图中红框圈起的部分进行配置修改。

    主要配置项

    两次 COMP 配置有细微差别，体现在 Basic Configuration 和 Advanced Configuration 中。其中某一个 COMP 的详细配置流程如下：

    TIMER-图1
    TIMER-图2

    另一 COMP 的配置与前一 COMP 的配置类似，区别如下：

    • Basic Configuration/Positive Terminal Channel Input 设为 COMPy positive terminal signal
    • Basic Configuration/Reference Voltage Generator Configuration/Input Code Through DACCODE0 设为较低值，但不要太低
    • Advanced Configuration/Enable Window Comparator 设为不使能
